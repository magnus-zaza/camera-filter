# Police Hat Filter - Patrol Mode Studio

A fun webcam filter that adds a police hat overlay to your face in real-time using face tracking technology.

## Features

- Real-time face tracking with MediaPipe Face Mesh
- Customizable badge name on the police hat
- Adjustable hat size
- Bilingual support (English & Arabic)
- Modern, responsive design
- Works directly in the browser - no installation required

## Usage

1. Open `index.html` in a modern web browser
2. Click "Start camera" and grant camera permissions
3. Enter a custom badge name (optional)
4. Adjust the hat size as needed
5. The hat will automatically track your face movements

## Technologies Used

- MediaPipe Face Mesh for face landmark detection
- Vanilla JavaScript for real-time rendering
- CSS3 for modern UI/UX
- Google Fonts (Cairo) for Arabic support

## Browser Requirements

- Modern browser with WebRTC support (Chrome, Firefox, Edge, Safari)
- Camera access permissions
- JavaScript enabled
